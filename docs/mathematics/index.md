# Mathematics

This is my map of the math I’m currently learning and organizing. The subjects overlap on purpose: number theory runs into analysis, representation theory opens the door to automorphic forms, and dynamics adds a geometric point of view.

## Areas I’m exploring

- [Analytic Number Theory](../analytic-number-theory/index.md): primes, \(L\)-functions, sieve methods, and analytic estimates
- [Probabilistic Number Theory](../probabilistic-number-theory/index.md): random models, distributions, and typical arithmetic behavior
- [Algebraic Number Theory](../algebraic-number-theory/index.md): number fields, local fields, ideals, class groups, and class field theory
- [Automorphic Forms](../automorphic-forms/index.md): automorphic representations, spectral theory, and arithmetic symmetry
- [Representation Theory](../representation-theory/index.md): groups, actions, harmonic analysis, and hidden structure
- [Homogeneous Dynamics](../homogeneous-dynamics/index.md): flows, invariant measures, equidistribution, and rigidity

!!! note

    These are living notes. Some pages are polished; others are still works in progress, full of questions I’m sorting out.
